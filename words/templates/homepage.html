What is MetaDictionary?

The purpose of this project is to have a dictionary that lists multiple definitions from different dictionary resources. Some dictionaries list the slang definitions of a word like Urban Dictionary, while others like Merriam-Webster list the formal definition of the word. By implementing multiple dictionary definitions together in one website, it can help users understand the context of the word.
